# moirai

the fates of what we build.

an interactive 3d experience that maps major construction projects worldwide onto a neon-holographic globe. click any project and dive from orbit to street level, where the neighborhood rebuilds as a glowing wireframe city — then watch the project's real impact (pollution, traffic, health, economy, habitat loss) ripple through the scene in real time.

## status

in active development. v0.1 (project scaffold) — done.

## stack

- vite + react 19 + typescript
- three.js (webgl globe + procedural city + particle effects)
- zustand (global state, added later)
- tailwind (styling, added later)
- framer motion (hud transitions, added later)
